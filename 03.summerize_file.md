# 03. Change Summary

## Phạm vi

File này tóm tắt các thay đổi tôi đã chủ động thực hiện trong repo để:
- sửa lỗi download / build retrieval DB
- thêm log để dễ debug
- sửa các lỗi Python phát sinh trong bước embedding
- giảm nguy cơ OOM
- hỗ trợ chạy embedding song song trên nhiều GPU

Không tính các file sinh tự động như `__pycache__`.

## 1. `script/download/build.sh`

### Đã thay đổi

- thêm cơ chế ghi log build vào:
  - `logs/build/<timestamp>/build_summary.log`
  - `logs/build/<dataset>__<step>.log`
- thêm chế độ resume:
  - skip `preprocess_raw_data` nếu TSV đã tồn tại
  - skip `generate_passage_embeddings` nếu shard embeddings đã hoàn tất
  - skip `build_index` nếu đã có index/docstore đầy đủ
- thêm cấu hình GPU và batch qua env:
  - `BUILD_CUDA_VISIBLE_DEVICES`
  - `BUILD_EMBED_BATCH_SIZE`
  - `BUILD_EMBED_TOTAL_SHARDS`
  - `BUILD_EMBED_SAVE_EVERY_BATCHES`
- đổi bước embedding từ 1 tiến trình / 1 GPU sang nhiều shard chạy song song
- mặc định dùng 4 GPU:
  - `2,3,4,5`
- mỗi shard có log riêng:
  - `...__generate_passage_embeddings__shard_00.log`
  - `...__shard_01.log`
  - ...

### Mục tiêu

- không phải chạy lại từ đầu khi lỗi giữa chừng
- tận dụng nhiều GPU để tăng tốc build embeddings
- dễ xác định shard nào lỗi

### Đã thay đổi

## 9. Cách chạy khuyến nghị

```bash
BUILD_CUDA_VISIBLE_DEVICES=2,3,4,5 BUILD_EMBED_BATCH_SIZE=64 bash script/download/build.sh
```

Nếu cần ép chạy lại toàn bộ:

```bash
BUILD_FORCE_REBUILD=1 BUILD_CUDA_VISIBLE_DEVICES=2,3,4,5 BUILD_EMBED_BATCH_SIZE=64 bash script/download/build.sh
```
